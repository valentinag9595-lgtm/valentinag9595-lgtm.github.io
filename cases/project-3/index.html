Anisad — Real Estate

Timeline 2024
Role Product Designer
Team Product Manager, Marketing Team, Developers

Problem

The existing website had poor navigation and an overloaded interface structure. Users found it difficult to quickly discover relevant properties and navigate through the catalog.

Additionally, the property listing submission flow was too long and complicated. Many users dropped off during onboarding and never published their listings.

Insight

For real estate platforms, it is critical that users:
  • quickly find relevant properties
  • receive clear and visually appealing information
  • complete key actions easily without unnecessary friction
The simpler the navigation and the clearer the property card structure, the higher the engagement, trust, and likelihood of registration or lead submission.

Reducing onboarding complexity also directly impacts the number of published listings on the platform.

Goal

Redesign the website experience to:
  • improve navigation
  • make property cards more modern and informative
  • increase user engagement
  • improve registrations and lead generation
  • simplify the property publishing flow

Research & Workshop

Together with the product and marketing teams, we analyzed user behavior and major drop-off points.

The research showed that:
  • users were getting lost in navigation
  • important property information was poorly structured
  • the interface looked visually outdated
  • the listing submission process contained too many steps and created frustration

Key Decisions

  • Simplify navigation and catalog structure
  • Completely redesign the visual style of property cards
  • Improve information hierarchy within property listings
  • Add more visible CTAs for registration and lead submission
  • Reduce the number of steps required to publish a property
  • Make the property submission onboarding flow faster and easier to understand

Implementation

As part of the redesign, we updated:
  • the homepage
  • the real estate catalog
  • property cards
  • registration flows
  • the onboarding flow for adding new listings
After several design iterations and testing rounds, we improved the visual structure, simplified user flows, and reduced the number of actions required to publish a listing.

Results

The new design helped:
  • increase the amount of time users spent on the website
  • improve registration rates
  • increase the number of leads and inquiries
  • improve overall user engagement

Impact

Additionally:
  • the property submission onboarding flow was simplified
  • users published listings faster
  • the number of properties on the platform increased
  • onboarding completion rate improved by 20%
